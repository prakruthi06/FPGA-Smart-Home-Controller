# FPGA-Based Smart Home Controller (HomeGuardX)

## Overview
This project implements a smart home automation system using Verilog HDL on FPGA.

## Features
- Temperature-based Fan Control
- Automatic Light Control
- Intruder Alert System (FSM-based)
- Configuration Registers for threshold control
- Modular and scalable architecture

## Tools Used
- Verilog HDL
- Xilinx Vivado
- Artix-7 FPGA

## System Architecture
The system consists of multiple modules:
- Configuration Registers
- Fan Controller
- Light Controller
- Intruder Alert Controller (FSM)
- Timer Module
- Top-Level Controller

## Results
- Successfully simulated using testbenches
- Synthesized and implemented on FPGA
- Generated valid bitstream

## Applications
- Smart Home Automation
- Security Systems
- Embedded FPGA Systems
